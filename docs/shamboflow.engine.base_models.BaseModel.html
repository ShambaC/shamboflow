View In Hierarchy

The Base class for models All other models need to inherit from this class. It provides a template with all the methods a model will need to execute. Attributes : ------------ layers : A list of layers present in the model in the order from input to output weights : A list of all weight matrices for the layers in the model loss : The loss/cost function to use calculate error of the model at a given state train_data_x : The training data or features train_data_y : Training features' corresponding labels and more
